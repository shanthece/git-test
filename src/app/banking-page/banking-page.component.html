Install Node

Test Node:

To see if Node is installed, using command Prompt or any command line tool, and type node -v or node --version. This should print a version number, so you’ll see something like this v6.11.2.

Test NPM:

To see if NPM is installed, type npm -v or npm --version. This should print a version number, so you’ll see something like this 3.10.10.

Then install the Angular CLI globally.

npm install -g @angular/cli

Create a new project:

ng new mydream-app

Serve the application

ng serve - It will open up in default browser, if it not opens use ng serve --open or http://localhost:4200/ open this url in your browser.
